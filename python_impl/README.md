# Fire Simulation - Python Implementation

This directory contains the Python implementation using **Numpy** for vectorized operations. It serves as the baseline for performance comparisons and demonstrates ease of implementation.

### Prerequisites
- **Python 3.8+**
- **Numpy**

### Setup
We recommend using a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate # for Windows: venv\Scripts\activate
pip3 install -r requirements.txt
```

### Run
Run the simulation script directly:

```bash
# Run with default settings
python3 main.py

# Run with custom parameters
python3 main.py --size 1000 --steps 100 --density 0.6
```
